# PixelToCode

PixelToCode is a web application that converts UI screenshots into clean, semantic HTML, CSS, and React code using AI.

## Features

- Upload screenshots of UI designs or websites
- AI analysis of visual elements and structure
- Generate clean HTML/CSS code
- Generate React/TypeScript code
- Copy or download generated code
- Light and dark mode support

## Tech Stack

- React
- TypeScript
- Vite
- TailwindCSS
- React Router
- React Syntax Highlighter
- React Dropzone

## Getting Started

### Prerequisites

- Node.js (v14 or later)
